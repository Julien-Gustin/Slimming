# By Gustin Julien & Sebati Ilias
## Slimming project

### How to use :
* Get into src/ repectory
* Type make
* Then
```sh
./slimming "input-file" "output-file" nbOfPixel-to-reduce
```

# Example of using

```sh
./slimming ../pnm/07.pnm ../pnmOutput/test07.pnm 500
```
